Pour la réalisation des tps nous avons été deux: Gygi Damien et Raphaël Schaffo
Tous les trois Tps ont été réalisé en OMP et en CUDA. 
Comme précisé lors du cours, comme l'implémentation Cuda fonctionne, nous n'avons pas créer les sh et run pour l'implémentation OMP ( comme on ne doit pas les rendre)
Les fichiers sh et run se trouve dans le dossier release:
	tp_rippling => ripplig
	tp_mandelbrot => mandelbrot
	tp_raytracing => raytracing

Comme demandé, nous avons aussi créer le projet démo qui lance les trois tps en même temps (le projet: Studend_Cuda_Image_Demo)
Nous avons également légérement décalé les fenêtres pour pouvoir bien différencier ces tps.
Nous avons aussi utiliser une forge. Vous pouvez retrouverez notre avancement en permance à cette adresse: https://github.com/DamienGygi/Cuda
